| North America | United States, Canada |
| Asia | Singapore |

| Europe (TARIC) | Austria	, Belgium, Bulgaria, Cyprus, Czech Republic, Germany, Denmark, Estonia, Greece, Spain, Finland, France, Croatia, Hungary, Ireland, Italy, Lithuania, Luxembourg, Latvia, Malta, Netherlands, Poland |

<Note>
  Only **import** HS Code is supported for all country-specific responses.
</Note>

## World Customs Organization (WCO)

If no country is specified in your request, the response defaults to the **WCO Harmonized System** - the international standard 6-digit HS code nomenclature maintained by the World Customs Organization.

We're constantly expanding our coverage. If you need a specific country that's not listed, please [contact us](mailto:support@hsping.com) and we'll prioritize it based on demand.
