HS Ping currently supports HS code lookups for the following countries, as well as the international format by WCO:

| 2-letter ISO Format | Supported Country |
| :------------------ | ----------------- |
| US                  | United States     |
| UK                  | United Kingdom    |
| SG                  | Singapore         |
| CA                  | Canada            |
| AU                  | Australia         |

<Note>
  Only **import** HS Code is supported for all country-specific responses.
</Note>

## World Customs Organization (WCO)

If no country is specified in your request, the API defaults to the **WCO Harmonized System** - the international standard 6-digit HS code nomenclature maintained by the World Customs Organization.

## Adding New Countries

We're constantly expanding our coverage. If you need a specific country that's not listed, please [contact us](mailto:support@hsping.com) and we'll prioritize it based on demand.
