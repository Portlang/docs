---
title: "Supported Countries"
---

HS Ping supports HS code references from the WCO (World Customs Organization) as well as the 5 countries below.

To search for country-specific HS code, use the <kbd>country</kbd> parameter with country value following the 2-letter ISO format. If no country parameter is specified in a request, the returning HS Code references would be based on the WCO 6-digits standard.

| Supported Country | 2-letter ISO Format | Full Country Parameter |
| ----------------- | ------------------- | ---------------------- |
| United States     | US                  | &country=us            |
| United Kingdom    | UK                  | &country=uk            |
| Singapore         | SG                  | &country=sg            |
| Canada            | CA                  | &country=ca            |
| Australia         | AU                  | &country=au            |

<Info>
  Only **import** HS Code is supported for all country-specific responses.
</Info>
