---
title: "Supported Countries"
---

You can search for country-specific HS code via the <kbd>country</kbd> parameter, using the 2 letters ISO country code format.

| Supported Country | 2 letters ISO country code | Full Country Parameter |
| ----------------- | -------------------------- | ---------------------- |
| United States     | US                         | &country=us            |
| United Kingdom    | UK                         | &country=uk            |
| Singapore         | SG                         | &country=sg            |
| Canada            | CA                         | &country=ca            |
| Australia         | AU                         | &country=au            |

<Info>
  Only import HS Code is supported for all country-specific responses.
</Info>

If no country parameter is specified in a request, HS Ping returns HS Code references based on the WCO (World Customs Organization) HS code standard.
