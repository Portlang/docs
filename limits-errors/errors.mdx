---
title: "Errors"
---

The HS Ping API uses string code to give more context and recommendations.

| Error Code           | Details                                                                                    |
| -------------------- | ------------------------------------------------------------------------------------------ |
| UNAUTHENTICATED      | Invalid or inactive API key                                                                |
| INVALID_PARAMETER    | Invalid request parameters                                                                 |
| RATE_LIMIT_EXCEEDED  | Too many requests. Slow down.                                                              |
| INSUFFICIENT_BALANCE | Your account balance is insufficient. Upgrade or try again when the free credit has reset. |
