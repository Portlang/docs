[HS Ping](https://hsping.com) is an API service that lets you search for country-specific HS Code references via our REST API endpoints. See [Supported Countries](/supported-countries) for our country coverage.

By providing keyword input, HS Ping returns HTTP responses that contain HS code references with matching keywords. All returning HTTP responses (including errors) are in JSON-formatted output.

The base API url is <kbd>https://api.hsping.com/api/v1</kbd> .

HS Ping's API supports single and bulk HS code search.

<Note>
  HS Ping is not a classification service and does not handle any classification.
</Note>
