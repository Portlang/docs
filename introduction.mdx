HS Ping is an API service that searches for country-specific HS Code references via REST API endpoints.

<Info>
  For our country coverage, see [Supported Countries](/supported-countries).
</Info>

The API lets you add instant HS code search into your software and workflow. All searches via the HS Ping API is performed within HS Ping's data, sourced from official sources.

You can read about our [data sources here](https://hsping.com/data-source).

<Note>
  HS Ping is NOT a classification service, and does not handle any product classification.

  We recommend talking to a licensed broker if you need help with classification.
</Note>

## How it works

To search for HS Code references, you need to send an API request with the search term (query) included. HS Ping would return a HTTP response that contain HS code references with matching query.

All returning HTTP responses (including errors) are in JSON-formatted output.

## Base API URL

The base API url is <kbd>https://api.hsping.com/api/v1</kbd> .

## Search Options

HS Ping's API supports single and bulk HS code search.
