---
title: "Query"
---

The <kbd>query</kbd> parameter is your search term for finding HS codes.

<ParamField path="query" type="string" required>
  The search term to initiate a HS code search
</ParamField>

<RequestExample>

``` filename
curl -X GET "https://api.hsping.com/api/v1/find?query=bamboo furniture" \
  -H "Authorization: Bearer API_KEY"

//query = bamboo furniture
```

</RequestExample>
